首先你需要打开 [Python的官网进行下载](https://www.python.org/downloads/windows/)，点击 `Latest Python 3 Release - Python 3.7.3` 进行下载：

![install on windows](./img/install_on_windows.png)

下载完成之后一路下一步，注意其中有一节需要点击 `Add Python To PATH`，把它勾上。

![install on windows step 1](./img/install_on_windows_step1.png)
![install on windows step 2](./img/install_on_windows_step2.png)
![install on windows step 3](./img/install_on_windows_step3.png)
![install on windows step 4](./img/install_on_windows_step4.png)
![install on windows step 5](./img/install_on_windows_step5.png)
![install on windows step 6](./img/install_on_windows_step6.png)

在cmd窗口输入 `python --version` 如果打印出了Python的版本是 `3.x` 那么就是ok的！

```bash
$ python --version
Python 3.6.0
```
