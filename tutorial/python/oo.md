---

Python中使用 `class` 定义类，Python支持多继承。

```python
In [1]: class Base:
   ...:     pass
   ...:

In [2]: class Children1(Base):
   ...:     pass
   ...:

In [3]: class Children2(Base):
   ...:     pass
   ...:

```
